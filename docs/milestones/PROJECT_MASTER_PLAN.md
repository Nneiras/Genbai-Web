# GENBAI - Plan Maestro de Proyecto

Este documento centraliza el estado actual, la visión y el seguimiento de las fases del proyecto GENBAI.

## Visión General
Transformar la presencia digital de GENBAI de una landing page estática a una plataforma inteligente con CRM integrado y agentes de IA especializados para Pymes.

## Roadmap de Fases

### Fase 1: Landing Page Premium (Completado)
- [x] Diseño moderno y responsive.
- [x] Implementación de modo claro/oscuro (Corregido posicionamiento).
- [x] Optimizaciones de SEO y rendimiento.
- [x] Galería de Blogs con Modal inteligente e imágenes actualizadas.

### Fase 2: CRM & Agentes de IA (En curso)
- [x] **CRM Base**: Integración con Supabase para captura de leads.
- [x] **Dashboard Admin**: Panel para gestión de clientes y métricas reales.
- [ ] **Agente de Atención**: Chat inteligente 24/7 en la web.
- [ ] **Agente de Contenido**: Generador de blogs y redes sociales.
- [ ] **Agente de Presupuestos**: Cotizador automático inteligente.
